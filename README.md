# YC-扬尘

一个基于微信小程序的生活助手应用，提供化妆品管理、财务管理、健康管理和衣柜管理等功能。

## 功能特性

### 化妆品管理
- 记录化妆品信息
- 管理化妆品库存
- 设置使用提醒

### 财务管理
- 记录收支明细
- 统计分析
- 预算管理

### 健康管理
- 健康数据记录
- 运动追踪
- 健康提醒

### 衣柜管理
- 衣物管理
- 搭配建议
- 购买记录

## 技术栈

- 微信小程序
- JavaScript
- WXML / WXSS

## 项目结构

```
├── i18n/              # 国际化配置
├── images/            # 图片资源
├── pages/             # 页面目录
│   ├── add/           # 添加页面
│   ├── cosmetics/     # 化妆品页面
│   ├── finance/       # 财务页面
│   ├── health/        # 健康页面
│   ├── history/       # 历史记录页面
│   ├── reminder/      # 提醒页面
│   └── wardrobe/      # 衣柜页面
├── app.js             # 应用入口
├── app.json           # 应用配置
├── app.wxss           # 全局样式
└── project.config.json # 项目配置
```

## 快速开始

1. 克隆项目到本地
2. 使用微信开发者工具打开项目
3. 配置小程序 AppID
4. 编译并预览

## 开发

```bash
# 克隆项目
git clone https://github.com/HUAZHAONIANER/YC-dust.git

# 进入项目目录

```
